# Expansion Panel

Expansion panel is a material Design component that is used to display a list of item that can be collapsed and expanded

# Run Instructions

- Clone the project ```git clone https://github.com/mbonikevin/ExpansionPanel-Widget-Presentation.git```
- And run the `Main.dart`

# note on three attributes

- `isExpande`: this attribute recives the state to check and update the ui of the pannel
- `headerBuilder`: This attribute holds text/elements that will be visible always even after collapsing
- `body`: all our texts/elements you want to be collapsed will go here in the body
